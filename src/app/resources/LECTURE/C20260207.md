C언어는 변수, 연산자, 제어문, 함수, 배열, 포인터, 구조체 로 이루어져 있다.

이 중 함수, 포인터, 구조체 를 기반으로 프로그래밍을 한다.

변수 : 변수는 값을 저장하는 공간이다
연산자 : 연산자는 변수의 값을 연산하는 기호이다
제어문 : 제어문은 프로그램의 실행 흐름을 제어하는 문법이다 (if-else if-else, for, while, do-while, switch, break, continue)
함수 : 함수는 특정 기능을 수행하는 코드의 묶음이다
배열 : 배열은 같은 타입의 변수들을 연속적으로 저장하는 공간이다
포인터 : 포인터는 변수의 주소를 저장하는 변수이다
구조체 : 구조체는 여러 타입의 변수들을 묶어서 저장하는 공간이다
